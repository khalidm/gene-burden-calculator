About BurGene

BurGene is a tool for prioritizing genes based on the number of observed genetic variants.

BurGene calculates the difference between the frequencies of selected variants in cases and reference populations such as the Exome Aggregation Consoritum (ExAC). The current version provides reference data from ExAC populations, with more refernce data from TCGA coming soon.

Methods:
The BurGene tools provides a pipeline that starts with preparing a VCF file for analysis by annotation and predicting variant impact using in-silico prediction methods. Next, we apply user specified variant level filters and based on the remaining variants calculate gene level burden of likely intereresting variants for further analysis. The same filters are then applied to the reference population data to form equivalent gene burdens.
We then apply hypothessis tests to compare the number of gene variants in the cases group to the reference dataset. The tests include two population proportion z-test and binomial exact tests. Also provided is a relative risk ratio.

Example usage:
You have germline exome data from 700 patients. You employed a standard variant calling pipeline (based on GATK) to call variants using hg19 as reference.
